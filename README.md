# 🏠 House Price Prediction using Gradient Boosting & XGBoost

## 📌 Project Overview
This project focuses on predicting house prices using advanced ensemble machine learning techniques such as Gradient Boosting and XGBoost. The goal is to build a robust regression model that captures complex relationships between housing features and prices.

---

## 🚀 Objectives
- Perform data preprocessing and feature engineering
- Build regression models using Gradient Boosting and XGBoost
- Compare model performance
- Improve prediction accuracy using ensemble methods

---

## 🧠 Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Matplotlib, Seaborn

---

## 📂 Dataset
- Bangalore House Price Dataset
- Includes features like:
  - Location
  - Size (BHK)
  - Total square feet
  - Number of bathrooms
  - Price

---

## ⚙️ Methodology

### 1. Data Preprocessing
- Handling missing values
- Feature engineering
- Encoding categorical variables
- Removing outliers

### 2. Model Building
- Gradient Boosting Regressor
- XGBoost Regressor

### 3. Model Evaluation
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

---

## 📊 Model Performance

| Model | Performance |
|------|------------|
| Gradient Boosting | Good baseline |
| XGBoost | Improved accuracy |

XGBoost performed better due to its regularization and ability to handle complex patterns.

---

## 📈 Key Insights
- Location and square footage are the most important features
- Ensemble models outperform basic regression models
- Proper preprocessing significantly improves performance

---

## 🔧 Improvements
- Hyperparameter tuning (GridSearchCV)
- Feature importance visualization
- Cross-validation
- Model deployment (Flask/Streamlit)

---

## 🎯 Conclusion
The project demonstrates the effectiveness of ensemble learning techniques like Gradient Boosting and XGBoost in solving regression problems. XGBoost achieved better generalization and accuracy, making it suitable for real-world applications.
